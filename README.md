# 📊 Análisis de Rentabilidad Regional - RappiPlus

Este repositorio contiene un análisis integral de rentabilidad para **RappiPlus**, abarcando operaciones en México, Colombia y Argentina. El objetivo principal fue diagnosticar la salud financiera de diferentes categorías de productos y optimizar la inversión en marketing.

## 🛠️ Stack Tecnológico
* **Python (Pandas/Matplotlib):** Procesamiento de datos y análisis estadístico.
* **Power BI:** Dashboard interactivo y visualización de KPIs.
* **Markdown:** Documentación técnica.

## 🚀 Puntos Clave del Proyecto
* **Limpieza de Datos:** Integración de 3 datasets (`orders`, `catalog`, `marketing`) asegurando la integridad referencial.
* **Análisis Geográfico:** Implementación de un *Heatmap* de rentabilidad para identificar mercados críticos.
* **KPIs Desarrollados:** Ticket Promedio, Unidades por Orden, Margen de Utilidad y ROI de Marketing.

## 📁 Estructura del Repositorio
* `notebooks/`: Análisis detallado paso a paso.
* `dashboard/`: Archivo .pbix para exploración interactiva.
* `data/`: Datasets utilizados en el modelo.
